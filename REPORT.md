## 4) Как открыть XML и сохранить в .mpp вручную

### Вариант A — Microsoft Project (рекомендуется)
1. Откройте Microsoft Project.
2. Выберите **File -> Open**.
3. Укажите файл `ЗАДАНИЕ_3_БАРАНОВА_ЭВЕЛИНА_ФИНАЛ.xml`.
4. При запросе импорта выберите открытие как проект MS Project XML.
5. Проверьте календарь, ресурсы, связи и факт в Gantt Chart / Resource Sheet / Task Usage.
6. Выберите **File -> Save As**.
7. Тип файла: **Project (*.mpp)**.
8. Сохраните рядом с XML, например: `ЗАДАНИЕ_3_БАРАНОВА_ЭВЕЛИНА_ФИНАЛ.mpp`.

### Вариант B — ProjectLibre
1. Откройте ProjectLibre.
2. **File -> Open** и выберите `...ФИНАЛ.xml`.
3. Проверьте корректность отображения задач/ресурсов/связей.
4. Сохраните в родной формат ProjectLibre (если MPP-экспорт недоступен в вашей версии).
5. Для именно `.mpp` используйте Microsoft Project (вариант A).
